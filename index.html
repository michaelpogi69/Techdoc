Introduction

Markdown is a lightweight markup language with plain-text formatting syntax.

Syntax

Headings

Use # for headings. For example:

# Heading 1
## Heading 2
### Heading 3

Lists

To create a list:

- Item 1
- Item 2
- Item 3

Examples

Here is an example of a Markdown document:

# My Document
This is a simple Markdown document.
- Point 1
- Point 2

Conclusion

Markdown is a simple and effective way to format text.
